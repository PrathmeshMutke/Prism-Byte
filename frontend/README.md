# Banking App Frontend

React + Vite + Tailwind frontend for the Spring Boot banking backend.

## Run locally

1. Open a terminal in `frontend`
2. Install packages:
   `npm install`
3. Start the dev server:
   `npm run dev`

The app expects the backend at `http://localhost:8080/api`.

## Environment

Create a `.env` file if you want to override the backend URL:

`VITE_API_BASE_URL=http://localhost:8080/api`

## Seeded credentials

- Employee: `employee@prismbyte.com / Employee@12345`
